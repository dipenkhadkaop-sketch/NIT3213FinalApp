# NIT3213 Android Application

## Overview
This project is an Android application developed for NIT3213. It demonstrates API integration, user interface design, and Android development best practices.

## Features
- Login screen with API integration
- Dashboard screen displaying data
- Details screen for item information
- Navigation between screens
- Dependency Injection using Hilt

## API
Base URL:
https://nit3213api.onrender.com/

Endpoints used:
- POST /auth
- GET /dashboard/{keypass}

## Note
During testing, the API returned errors, so a fallback navigation was implemented to demonstrate full app functionality.

## How to Run
1. Open the project in Android Studio
2. Sync Gradle
3. Run the app on emulator or device
